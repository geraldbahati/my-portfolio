# Security Policy

## Supported Versions

Only the latest version deployed from `main` is supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

- **Email:** contact@geraldbahati.dev
- **GitHub:** Use [private vulnerability reporting](https://github.com/geraldbahati/my-portfolio/security/advisories/new)

Please do not open public issues for security vulnerabilities.

## Response

I will make a best effort to acknowledge reports within 7 days and provide a fix or mitigation as soon as possible.
